# 🌱 About Kallali

**Kallali** is an open-source full-stack web platform designed to empower farmers by enabling them to sell agricultural products directly to buyers without relying on intermediaries.

The project combines modern web technologies with scalable backend architecture and is being developed publicly as a real-world software engineering journey.

The long-term vision is to evolve Kallali into a complete AgriTech ecosystem featuring an AI-powered farming assistant, intelligent product recommendations, secure payments, and data-driven farming insights.

# 🎯 Problem Statement

Agriculture remains one of the world's most important industries, yet many farmers still struggle with challenges such as:

- Limited access to direct buyers
- Heavy dependence on middlemen
- Lack of transparent market pricing
- Difficulty managing products digitally
- Limited access to modern farming guidance

Kallali aims to solve these problems by providing a secure digital marketplace that connects farmers and buyers while leveraging AI to support smarter agricultural decisions.

# 🌐 REST API

## Authentication Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| POST | `/api/auth/signup` | Register a new user |
| POST | `/api/auth/login` | Authenticate a user |
| GET | `/api/auth/me` | Get authenticated user profile |
| GET | `/api/auth/admin` | Protected admin route |
| GET | `/api/healthz` | Health check |

---

# 🔐 Authentication Flow

```text
                 ┌──────────────┐
                 │    Signup    │
                 └──────┬───────┘
                        │
                 Validate Request
                        │
                 Hash Password
                        │
                 Store in PostgreSQL
                        │
                 Registration Success

──────────────────────────────────────────

                 ┌──────────────┐
                 │     Login    │
                 └──────┬───────┘
                        │
                 Verify Credentials
                        │
                 Generate JWT
                        │
                 Return Token
                        │
                 Access Protected Routes

──────────────────────────────────────────

                 Protected Request
                        │
                 JWT Middleware
                        │
                 Verify Token
                        │
                 Role Authorization
                        │
                 Allow / Deny
```

# 🚀 Getting Started

## Prerequisites

Before running the project, make sure you have:

- Node.js (Latest LTS)
- PostgreSQL
- pnpm

---

## Clone Repository

```bash
git clone https://github.com/pasmulwaryogesh96-boop/kallali.git

cd kallali
```

---

## Install Dependencies

```bash
pnpm install
```

---

## Configure Environment Variables

Create your environment files.

Backend example:

```env
DATABASE_URL=your_database_url

JWT_SECRET=your_secret_key
```

---

## Run Backend

```bash
cd artifacts/api-server

pnpm dev
```

---

## Run Frontend

```bash
cd artifacts/kallali

pnpm dev
```
